================================================================================
DATABASE STRUCTURE
================================================================================

Database File: ~/.nex-deliverables/deliverables.db (SQLite 3)

Tables (4):
  1. clients - Client master data
  2. deliverables - Project deliverables tracking
  3. milestones - Sub-tasks within deliverables
  4. status_updates - Status change history

Indexes (5):
  - deliverables(client_id)
  - deliverables(status)
  - deliverables(deadline)
  - clients(status)
  - milestones(deliverable_id)

================================================================================
COMMAND REFERENCE - 12 COMMANDS TOTAL
================================================================================

1. ADD         - Add new deliverable
   Handler:    cmd_add()
   Args:       --client, --title, --type, [--deadline, --description, --priority, ...]

2. CLIENT      - Manage clients
   Handler:    cmd_client()
   Subcommands: add, show, list
   Args:       Varies by subcommand

3. LIST        - List deliverables with filters
   Handler:    cmd_list()
   Args:       [--client, --status, --type, --priority, --overdue]

4. SHOW        - View deliverable details
   Handler:    cmd_show()
   Args:       deliverable (ID or title)

5. STATUS      - Update deliverable status
   Handler:    cmd_status()
   Args:       deliverable_id, status, [--message]

6. MARK        - Quick mark by title
   Handler:    cmd_mark()
   Args:       title, status

7. OVERDUE     - Show overdue deliverables
   Handler:    cmd_overdue()
   Args:       None

8. SEARCH      - Full-text search
   Handler:    cmd_search()
   Args:       query

9. WORKLOAD    - Workload summary
   Handler:    cmd_workload()
   Args:       None

10. EMAIL      - Generate status email
    Handler:   cmd_email()
    Args:      client

11. EXPORT     - Export to CSV or JSON
    Handler:   cmd_export()
    Args:      [--format, --client, --output]

12. STATS      - Delivery statistics
    Handler:   cmd_stats()
    Args:      None

QUICK START
================================================================================

1. Extract to ~/nex-deliverables/

2. cd ~/nex-deliverables/ && bash setup.sh

3. Add a client:
   nex-deliverables client add --name "Acme" --email "contact@acme.com"

4. Add a deliverable:
   nex-deliverables add --client "Acme" --title "Website" --type website

5. View all:
   nex-deliverables list

For complete documentation: See SKILL.md
